# **App Name**: MedStock Manager

## Core Features:

- Offline Sales: Offline Sales Recording: Record sales transactions even without an internet connection.
- Inventory: Inventory Tracking: Monitor stock levels, set reorder points, and receive alerts for low stock.
- Purchase Orders: Purchase Order Management: Create and manage purchase orders for restocking.
- Returns: Returns Handling: Process and track customer returns, updating inventory accordingly.
- Smart Restock: Smart Restock tool: Suggests optimal order quantities based on sales trends and expiration dates. The tool can analyze historical data and current stock levels to prevent shortages and overstocking.
- Reports: Reporting Dashboard: Generate sales reports, inventory reports, and purchase order reports.
- Mobile Access: Mobile Accessibility: Ensure the application is fully responsive and functional on both desktop and mobile devices.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, security, and reliability in healthcare management.
- Background color: Light gray (#ECEFF1), a desaturated version of the primary, to maintain a clean and professional look without distracting from key data.
- Accent color: Soft green (#8BC34A), analogous to blue, used for positive confirmations and available stock indications to add a touch of reassurance.
- Body and headline font: 'PT Sans' (sans-serif) for clear and accessible information display across all devices, promoting readability and a modern user experience.
- Use clear and recognizable icons for all primary functions like sales, inventory, and reports.
- Implement a responsive grid layout for adaptability on both desktop and mobile devices.
- Incorporate subtle transition animations to improve user experience and guide users through different processes smoothly.
